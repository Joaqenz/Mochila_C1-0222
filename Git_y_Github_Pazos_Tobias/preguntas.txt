a. ¿Quién inventó el sistema de control de versión Git y por qué?
    El creador de Git es Linus Torvalds y fue creado para poder llevar registro de los cambios en archivos incluyendo coordinar el trabajo que varias personas realiza sobre archivos compartidos.
